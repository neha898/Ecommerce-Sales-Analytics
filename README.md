# E-Commerce Sales Analytics Dashboard

## Project Overview
This project analyzes e-commerce sales data using SQL, Python, and Power BI to uncover business insights related to sales performance, profitability, customer segments, and regional trends.

---

## Tools Used
- SQL
- Python
- Pandas
- Matplotlib
- Seaborn
- Power BI

---

## Key Business Insights
- Technology category generated highest revenue.
- West region contributed maximum profit.
- Consumer segment generated highest sales.
- Higher discounts negatively impacted profitability.
- Standard Class was the most preferred shipping mode.

---

## Project Workflow
1. Data Cleaning using Python
2. SQL-based Business Analysis
3. Exploratory Data Analysis (EDA)
4. Interactive Power BI Dashboard Creation
5. Business Insight Generation

---

## Dashboard Features
- KPI Cards
- Sales Analysis
- Profit Analysis
- Region-wise Insights
- Discount vs Profit Analysis
- Interactive Slicers
